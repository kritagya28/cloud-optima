IAM Policy Verification

Attach this least-privilege policy to your IAM user for read-only metadata access.

{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Action": [
        "ce:GetCostAndUsage",
        "cloudwatch:GetMetricStatistics",
        "cloudwatch:ListMetrics",
        "ec2:DescribeInstances",
        "s3:GetBucketTagging",
        "ebs:DescribeVolumes"
      ],
      "Resource": "*"
    }
  ]
}
